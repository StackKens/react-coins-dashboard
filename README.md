# React Coins Dashboard

A practice React application built with **Vite**, **Tailwind CSS**, and **React Router** to learn API integration and client-side routing using live cryptocurrency data.

## Purpose

This project is focused on:

- Fetching and displaying data from a public coins API
- Practicing React Router for page navigation
- Building responsive layouts with Tailwind CSS
- Understanding real-world frontend application structure

## Tech Stack

- React
- Vite
- Tailwind CSS
- Public Crypto API (CoinGecko)
